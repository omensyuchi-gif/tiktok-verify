OMEN'S Marketing Dashboard

メンズ眉毛専門サロン「OMEN'S」の社内マーケティング分析ツール

このアプリについて

本アプリは、株式会社Growth Sence が運営するメンズ眉毛専門サロン「OMEN'S」が、自社の公式SNSアカウントのアナリティクスを社内ダッシュボードで可視化するために利用する、社内向けのマーケティング分析ツールです。

各SNSプラットフォームが提供する公式API（TikTok Display API 等）を通じて、当サロン自身が保有・管理する公式アカウントに限り、以下のデータを取得します。

  • 公式アカウントのプロフィール情報（フォロワー数・いいね合計数・投稿数 等）
  • 当サロンが公開した投稿の統計情報（再生数・いいね数・コメント数・シェア数 等）

取得したデータは社内のマーケティング分析・可視化の目的にのみ利用し、第三者へ公開・提供することはありません。第三者の個人アカウントのデータを取得することはありません。
